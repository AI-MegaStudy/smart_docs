# DL 신선도 판별 기획서

## 1. 목적

DL은 점주가 수확 후 과일을 선별할 때 사진 기반 품질 판별을 보조한다. 최종 출고 판단은 점주가 확정한다.

## 2. 입력/출력

입력:

```text
발주 품목 ID
과일 이미지
```

출력:

| 출력 | 저장 컬럼 |
|---|---|
| 모델 예측 등급 | `quality_inspections.model_grade` |
| 신선도 점수 | `quality_inspections.freshness_score` |
| 색상 점수 | `quality_inspections.color_score` |
| 둥근 정도 점수 | `quality_inspections.roundness_score` |
| 멍 확률 | `quality_inspections.bruise_probability` |
| 모델 판정 | `quality_inspections.model_decision` |
| 점주 확정 등급 | `quality_inspections.owner_confirmed_grade` |
| 점주 확정 판정 | `quality_inspections.owner_decision` |

## 3. 모델 전략

| 구성 | 설명 |
|---|---|
| PyTorch CNN | 이미지 등급 분류 |
| OpenCV 보조 분석 | 색상, 둥근 정도, 멍 후보 영역 계산 |
| 점수 통합 | 신선도 점수와 모델 판정 계산 |

추천 모델:

```text
ResNet18 또는 MobileNetV3 전이학습
```

## 4. API 흐름

```text
점주 앱 이미지 촬영
→ Multipart 업로드
→ FastAPI 저장
→ PyTorch inference
→ quality_inspections 저장
→ 점주 앱 결과 표시
→ 점주 최종 판정 저장
```

## 5. 데이터 전략

| 데이터 | 활용 |
|---|---|
| 농산물 품질(QC) 이미지 | 등급 분류 학습 |
| 전북 장수 사과 당도 품질 데이터 | 사과 품질 보조 학습 |
| 팀 자체 촬영 이미지 | 데모 검증 |

## 6. 신선도 점수 예시

```text
freshness_score = CNN 품질 확률 기반 점수 * 0.6
                + color_score * 0.2
                + roundness_score * 0.1
                + (1 - bruise_probability) * 100 * 0.1
```
