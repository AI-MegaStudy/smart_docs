# 점주 관리 앱 기획서

## 1. 목적

점주 관리 앱은 과수농가 점주가 상품, 수확 슬롯, 고객 주문, 발주 승인, 신선도 검사, 배송, 반품/환불을 관리하기 위한 Flutter Android 앱이다.

## 2. 점주 업무 흐름

```text
로그인
→ 농장 정보 등록
→ 상품 등록
→ ML 예측 실행
→ 수확 슬롯 확정
→ 고객 예약/주문 확인
→ 발주 승인
→ 신선도 검사
→ 배송 등록
→ 반품/환불 처리
```

## 3. 주요 기능

| 기능 | 설명 | 관련 테이블 |
|---|---|---|
| 점주 로그인 | 점주 권한 계정 로그인 | `accounts`, `owner_profiles` |
| 농장 관리 | 농장명, 지역, 주소, 정책 관리 | `farms` |
| 상품 관리 | 상품명, 품종, 포장 단위, 가격, 이미지 관리 | `products` |
| ML 예측 | 수확 예정 범위, 수확량, 추천가 확인 | `ml_predictions` |
| 수확 슬롯 확정 | 고객에게 노출할 수확 예정 범위, 수량, 가격 확정 | `harvest_slots` |
| 예약/주문 현황 | 고객 예약과 주문 상태 확인 | `reservations`, `orders` |
| 발주 관리 | 발주 승인/부분승인/거절 | `procurements`, `procurement_items` |
| 신선도 검사 | 과일 이미지 촬영 후 DL 결과 저장 | `quality_inspections` |
| 배송 관리 | 송장번호, 발송 수량, 배송 상태 등록 | `shipments` |
| 반품/환불 관리 | 반품 요청 결정, 환불 처리 | `return_requests`, `refunds`, `payments` |

## 4. ML 사용 정책

ML 예측 결과는 점주의 의사결정 보조 정보다.

점주 앱에는 다음 값을 보여준다.

- 예측 수확 시작일/종료일
- 예상 수확량 kg
- 제안 예약 가능 최소/최대 kg
- 추천 가격
- 신뢰도
- 안전 계수
- 주의 메시지

점주는 이 값을 보고 직접 아래 값을 확정한다.

- 점주 확정 수확 시작일/종료일
- 점주 확정 예약 가능 kg
- 점주 확정 판매가
- 고객 고지 문구

## 5. DL 사용 정책

DL 신선도 판별 결과는 자동 출고 판단이 아니다. 점주 앱은 모델 결과와 점주 최종 판정을 분리한다.

| 구분 | 컬럼 |
|---|---|
| 모델 결과 | `model_grade`, `freshness_score`, `bruise_probability`, `model_decision` |
| 점주 확정 | `owner_confirmed_grade`, `owner_decision` |

## 6. 점주 대시보드 지표

| 지표 | 계산 기준 |
|---|---|
| 열린 수확 슬롯 수 | `harvest_slots.slot_status=OPEN` |
| 신규 발주 수 | `procurements.procurement_status=REQUESTED` |
| 선별 대기 품목 | 승인된 `procurement_items` 중 검사 미등록 |
| 배송 준비 주문 | `orders.order_status=READY_TO_SHIP` |
| 반품 요청 수 | `return_requests.return_status=REQUESTED` |
