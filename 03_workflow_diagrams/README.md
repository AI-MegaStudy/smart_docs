# Harvest Slot 워크플로우 다이어그램

고객 예약웹과 점주앱의 화면 단위 유저플로우를 시각화한 HTML 다이어그램입니다. 전체 서비스 흐름도가 아니라, 개발자와 디자이너가 각 화면 안에서 확인해야 할 입력, 판단, 처리, DB 사용 지점을 화면별로 분리했습니다.

도형 의미는 표준 플로우차트 관례를 따릅니다. 시작/종료는 둥근 사각형, 처리는 사각형, 판단 분기는 마름모, 입출력은 평행사변형으로 표시했습니다. DB가 사용되는 구간은 일반 원통형 아이콘으로 표시하고, 어떤 데이터가 조회/생성/갱신되는지 함께 적었습니다.

## 실행 방법

VS Code 확장 프로그램 **Live Server**로 실행합니다.

확장 링크: https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer

## Live Server 설치

1. VS Code를 실행합니다.
2. 왼쪽 사이드바에서 Extensions 아이콘을 누릅니다.
3. 검색창에 `Live Server`를 입력합니다.
4. 제작자가 `Ritwick Dey`인 확장 프로그램을 선택합니다.
5. `Install`을 누릅니다.

터미널에서 설치할 수도 있습니다.

```bash
code --install-extension ritwickdey.LiveServer
```

## 사용법

1. VS Code에서 이 README가 들어 있는 폴더 또는 상위 프로젝트 폴더를 엽니다.
2. `index.html` 파일을 엽니다.
3. HTML 편집 화면에서 우클릭합니다.
4. `Open with Live Server`를 선택합니다.
5. 브라우저에서 상단 탭을 눌러 다이어그램을 확인합니다.

이 다이어그램은 정적 HTML입니다. 별도 백엔드 실행이나 데이터베이스 연결은 필요하지 않습니다.

## 구성

- `index.html`: 화면별 유저플로우 다이어그램
- `workflow_customer.png`: 고객 예약웹 유저플로우 스크린샷
- `workflow_owner.png`: 점주앱 유저플로우 스크린샷
- `workflow_state.png`: 주문 처리 상태 전이 스크린샷
- `workflow_full.png`: 고객 예약웹 대표 스크린샷

## 포함 다이어그램

- 고객 예약웹: 상품 탐색, 상품 상세, 예약함, 예약 확인, 계정 인증, 주문서 작성, 결제, 주문 확인, 반품 신청
- 점주앱: 로그인, 대시보드, 농장 관리, 상품 관리, 상품 등록, 수확 예측, 수확 슬롯 확정, 발주 결정, 신선도 검사, 배송 관리, 반품·환불 관리, 내 정보
- 상태 전이: 예약 생성, 결제 승인, 발주 결정, 신선도 검사, 배송 등록, 배송 완료, 반품 결정
