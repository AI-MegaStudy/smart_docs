# WBS 및 역할 분담 2주

## 1. 인원 구성

팀장 1명과 실무 7명 기준으로 역할을 분담한다.

| 역할 | 주요 책임 | 산출물 |
|---|---|---|
| 팀장/기획 | 일정 관리, 문서 통합, 발표 총괄 | 기획서, 발표자료, WBS, 리스크 관리 |
| 백엔드 1 | 인증, 상품, 수확 슬롯 API | Auth/Product/Slot API |
| 백엔드 2 | 예약, 주문, 결제, 발주 API | Reservation/Order/Payment/Procurement API |
| DB/데이터 | MySQL, DDL, Open API 수집, 시드 데이터 | DDL, Seed, 데이터 수집 코드 |
| 사용자 웹 | Flutter Web 고객 화면 | 고객 예약 웹 |
| 점주 앱 | Flutter Android 점주 화면 | 점주 관리 앱 |
| ML 담당 | 수확 예측 모델 | ML 모델, 예측 API 연결 |
| DL/QA 담당 | PyTorch 신선도 판별, 통합 테스트 | DL 모델, QA 시나리오, 시연 점검 |
