# 디자인 시스템 및 UIUX 가이드

## 1. 디자인 컨셉

Harvest Slot은 신선함, 농가 신뢰, 예약 안정성을 강조한다.

키워드:

```text
신선함
정직한 농가
예약 안정성
수확 직배송
명확한 상태 안내
```

## 2. 컬러 토큰

| 토큰 | 용도 |
|---|---|
| Primary Green | 농장, 신선함, 주요 CTA |
| Warm Yellow | 수확, 예약 강조 |
| Neutral Gray | 배경, 구분선, 본문 |
| Alert Red | 반품, 거절, 주의 상태 |
| Info Blue | 예측 참고, 상태 안내 |

## 3. 공통 컴포넌트

| 컴포넌트 | 설명 |
|---|---|
| `StatusBadge` | 주문/발주/배송 상태 표시 |
| `NoticeBox` | 수확 일정 안내와 고지 문구 표시 |
| `ProductCard` | 상품 목록 카드 |
| `HarvestSlotCard` | 수확 슬롯 선택 카드 |
| `OwnerStatusCard` | 점주 대시보드 지표 카드 |
| `QualityResultCard` | DL 신선도 결과 표시 |
| `FormSection` | 입력 폼 구획 |

## 4. 고객 웹 UX 원칙

1. 상품 상세에서 수확 예정 범위와 고지 문구를 명확히 표시한다.
2. 예약 가능 수량은 kg와 포장 수량을 함께 표시한다.
3. 예약 확인 화면에서 수량과 가격을 다시 보여준다.
4. 주문 상세 화면은 상태 타임라인을 사용한다.
5. 반품 신청 화면은 사유 입력과 증빙 이미지 첨부 흐름을 단순하게 만든다.

## 5. 점주 앱 UX 원칙

1. 대시보드는 오늘 처리할 발주와 반품을 먼저 보여준다.
2. ML 결과와 점주 확정 입력을 같은 화면에 배치한다.
3. 발주 상세 화면은 승인 수량을 쉽게 조정할 수 있어야 한다.
4. 신선도 검사 화면은 촬영, 결과, 확정을 한 흐름으로 구성한다.
5. 배송 등록은 송장번호와 발송 수량 입력을 중심으로 단순화한다.

## 6. 문구 원칙

고객 화면:

```text
농가가 확정한 수확 예정 범위입니다.
기상과 생육 상황에 따라 일정이 조정될 수 있습니다.
```

점주 화면:

```text
ML 예측은 참고 자료입니다. 실제 예약 오픈 수량은 점주가 확정합니다.
DL 판별은 선별 보조 결과입니다. 최종 판정은 점주가 확정합니다.
```
