# Harvest Slot 점주앱 프로토타입

과수농가 점주가 수확 예측, 예약 오픈, 발주 승인, 선별, 배송, 반품을 처리하는 모바일 앱 고충실도 목업입니다.

화면 문구는 점주가 실제로 이해할 업무 표현으로 정리했으며, 백엔드/API/DB/개발자용 상태 코드가 화면에 노출되지 않도록 구성했습니다.

## 실행 방법

VS Code 확장 프로그램 **Live Server**로 실행합니다.

확장 링크: https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer

설치 상태: `ritwickdey.LiveServer` 확장이 이미 설치되어 있음을 확인했습니다.

## Live Server 설치

### VS Code 화면에서 설치

1. VS Code를 실행합니다.
2. 왼쪽 사이드바에서 Extensions 아이콘을 누릅니다.
3. 검색창에 `Live Server`를 입력합니다.
4. 제작자가 `Ritwick Dey`인 확장 프로그램을 선택합니다.
5. `Install`을 누릅니다.

### 명령어로 설치

VS Code 명령어 도구가 설정되어 있다면 터미널에서 아래 명령어를 실행합니다.

```bash
code --install-extension ritwickdey.LiveServer
```

## 사용법

1. VS Code에서 이 README가 들어 있는 폴더 또는 상위 프로젝트 폴더를 엽니다.
2. `index.html` 파일을 엽니다.
3. HTML 편집 화면에서 우클릭합니다.
4. `Open with Live Server`를 선택합니다.
5. 브라우저가 자동으로 열리면 화면을 확인합니다.

Live Server 기본 주소는 보통 아래 형식입니다.

```text
http://127.0.0.1:5500/index.html
```

포트가 이미 사용 중이면 Live Server가 다른 포트를 사용할 수 있습니다. 브라우저에 열린 주소를 그대로 사용하면 됩니다.

이 프로토타입은 정적 HTML입니다. 별도 백엔드 실행이나 데이터베이스 연결은 필요하지 않습니다.

## 메인 파일

파일: `index.html`

상단 필터로 화면 묶음을 전환할 수 있습니다.

- 전체
- 기본 관리
- 수확 운영
- 주문 처리
- 배송·반품

## 화면별 설명

- 로그인: 점주 계정으로 앱에 접속하는 첫 화면
- 대시보드: 오늘 처리해야 할 발주, 선별, 배송, 반품 업무 요약
- 농장 관리: 농장명, 주소, 소개, 배송 정책 관리
- 상품 관리: 판매 중인 상품 목록과 상태 확인
- 상품 등록: 상품명, 과일 종류, 품종, 포장 단위, 가격, 설명, 대표 이미지 등록
- 수확 예측: 수확 범위, 예상 수확량, 권장 예약량, 권장 판매가 확인
- 수확 슬롯 확정: 고객에게 열 수확 기간, 예약 가능 수량, 판매가, 안내 문구 확정
- 예약·주문 현황: 고객 예약과 주문 상태 확인
- 발주 목록: 결제 후 생성된 출고 처리 요청 확인
- 발주 상세: 승인, 부분 승인, 거절 결정
- 신선도 검사: 과일 촬영, 판별 결과 확인, 점주 최종 판정
- 배송 관리: 택배사, 송장 번호, 발송 수량 등록
- 반품·환불 관리: 반품 요청 확인, 승인 금액과 결정 사유 입력
- 내 정보: 점주 이름, 연락처, 사업자번호, 알림 설정 확인
